# KhushiDD

Componente de campo de texto de correo electrónico, diseñado para entradas de email.

#### Default Example

<Canvas of={KhushiDD.Default} />

## Usage

```ts
import { KhushiDD } from 'milibreria';
import { useState } from 'react';

function Example() {
    const [email, setEmail] = useState("");

    return (
        <KhushiDD
            placeholder="Ingrese su correo electrónico"
            value={email}
            onChange={(e) => setEmail(e.target.value)}
            size="small"
            height="40px"
            width="100%"
        />
    );
}
