# MyFavourite

Componente de botón de icono de corazón para marcar como favorito.

#### Primary Example

<Canvas of={MyFavourite.Primary} />

## Usage

```tsx
import MyFavourite from 'milibreria';

function Example() {
    return (
        <MyFavourite
            color="red"
            onClick={() => alert("Icono Favorito activado")}
        />
    );
}

export default Example;
```
