# MyFavouriteBorder

Componente de botón de icono de corazón contorneado para marcar o desmarcar como favorito.

#### Primary Example

<Canvas of={MyFavouriteBorder.Primary} />

## Usage

```ts
import { MyFavouriteBorder } from 'milibreria';

function Example() {
    return (
        <MyFavouriteBorder
            color="blue"
            onClick={() => alert("Corazón contorneado activado")}
        />
    );
}

export default Example;
```
